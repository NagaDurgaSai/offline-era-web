# Offline Era — Website

Official website for [Offline Era](https://github.com/NagaDurgaSai/offline_era), a peer-to-peer LAN messenger built for teams that work without internet.

Built with Next.js 14, TypeScript, and Tailwind CSS. Hosted on GitHub Pages.

## Stack

- Next.js 14 (static export)
- TypeScript
- Tailwind CSS v4
- Styrene A — typeface by Klim Type Foundry

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs to `out/` — ready for GitHub Pages.
